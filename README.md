# one-page-portfolio

Single-file HTML portfolio for Caitlin C. Nielson — senior UX researcher, fintech + AI.

The full portfolio lives on Framer. This is the lightweight one-pager: editorial layout, file-system nav conceit, built to share from a meeting or an email.

## Stack
- Hand-written HTML + CSS, no framework
- Fonts: Fraunces, Space Grotesk, Space Mono (Google Fonts)
- Palette: warm bone, charcoal ink, Bondi Blue primary, coral secondary
- Targets WCAG 2.1 AA

## Run
Open `index.html` in any browser.
