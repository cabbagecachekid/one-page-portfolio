## Design direction for Claude Code

- **Aesthetic:** Editorial/refined. Think a well-designed magazine or research journal — not a typical UX portfolio template.
- **Color:** Light background. One strong accent color (not purple gradient). High contrast type.
- **Typography:** A distinctive display font for the hook + headings. Clean readable body font. NOT Inter or Roboto.
- **Layout:** Generous whitespace. Left-aligned text feels more editorial than centered.
- **No animations needed** — this is a reading experience, not a demo.
- **Dark mode optional** — skip it for now, keep it simple.

---

## Recommended changes from original plan

1. **Fintech first** — the original hook options were infrastructure-forward. Flip the order: domain depth leads, infrastructure is the second beat.

2. **Label the AI work explicitly** — the Intuit funnel is an AI research story. Call it that on the page. Don't make a fintech recruiter infer it.

3. **"Make peace with AI" → "navigate AI"** — more accurate to what you actually do (behavior research, usability, trust) and less passive.

4. **One page, not a full site** — for tomorrow's meeting, a single polished HTML file is better than a half-built multi-page portfolio. Ship the one thing.

5. **File-system nav is a Phase 2 idea** — great for the full Framer build, skip it for the HTML page.
